Небольшая подсказка, switch работает двумя способами:
1. Сравнивает переданное значение, с определенными значениями, например:
let value = 2;
switch (value) {
    case 1:
    case 2:
    default:
}
2. Сравнивает значение с условиями, например:
let value = 2;
switch (true) { //То есть ищет кейс, при котором у нас получится true
    case value < 4:
    case value > 4:
    default: // если значение равно 4
}
