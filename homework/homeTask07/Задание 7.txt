Третье задание из практики в классе. HTML код для задания взять из того же архива
1) Создайте функцию onPageLoad, которая будет запускаться при загрузки странице.
	1) Создайте переменную userStateWrapper, в неё поместите DOM элемент #user-state
	2) Добавить для document обработчик событий, который при срабатывании DOMContentLoaded запускает функцию onPageLoad
	3) При запуске функции onPageLoad проведите проверку, есть ли в Cookie сохранённая запись sessionId, со значением
		2.1) Если такое значение есть - в userStateWrapper замените текст на Авторизованный пользователь
		2.2) Если значения нет - вывести Confirm на экран, с вопросом "Хотите авторизоваться?".
		При положительном ответе - Нужно добавить запись sessionId в куки, со следующим значением и параметром, после чего перезагрузить страницу.
		Значение: одной строкой, без пробелов ГодМесяцЧислоЧасМинута
		Параметр: время жизни этой записи - 1 минутаs
