Выполнить практическое задание из работы в классе по HTML форме.
Результат, которого нужно добиться - скриншот Результат 3.

Для каждого текстового поля и выпадающего списка возьмите за пример имеющийся элемент:
	Обёртка input-wrapper. Для каждого поля одна такая обёртка.
		Обёртка input-label. Внутри неё будет элемент label.
		Обёртка input-field. Внутри неё будет элемент input.



Для флажка и кнопки - несколько другая структура:
	Обёртка input-wrapper по прежднему имеется. Но внутри неё больше не надо обёртки input-label и input-field.
	Внутри неё сразу добавляйте label и input, либо button (либо input типа кнопка).

Для переключателей - не нужно обёртки input-wrapper.
	Сразу используйте fieldset.
		Добавьте подпись legend.
		Также, внутри него добавьте нужные label и input.

Для всех полей добавьте корректные атрибуты и их значения из следующих:
	type - тип поля, соответствующий информации, запрашиваемой этим полем
	placeholder - подпись для поля, подсказка
	id - индентификатор, чтобы связать label поля с полем (только там, где нужен)
	name - на английском, коректный заголовок поля, который будет передаваться на бэкенд
	required - для полей, обязательных к заполнению (все поля, кроме телефона, обязательны)
	value - значение поля, либо опции выпадающего поля, передаваемые на бэкенд (не нужно для текстовых полей). Для стран используйте двубуквенные коды стран.
	selected/checked - предвыбранные поля (например, страна по умолчанию, либо галочка для поля)

Обязательно свяжите все label и input.
