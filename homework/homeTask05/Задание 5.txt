Используйте объект Магазин, созданный в прошлой домашней работе, и на основе этого объекта создайте:
(Ниже перечислены свойства и методы по примеру прошлого задания, если вам для работы требуется большее количество методов - смело их добавляйте)

	1) Класс Магазин, у которого будут:
		Публичные методы:
		1) Добавить продукт
			Выбрать категорию и добавить в неё продукт
		2) Удалить продукт
			Ввести имя продукта, найти его и удалить
		3) Установить цену
			Получить как параметр, либо выбрать из ассортимента методом Выбрать продукт, продукт, и указать ему цену
		4) Установить количество
			Получить как параметр, либо выбрать из ассортимента методом Выбрать продукт, продукт, и указать ему количество
		5*) Пришёл покупатель
			Создать экземпляр класса Покупатель, и вызвать его метод Совершить покупку. После этого добавить его в Список покупателей
		6*) Показать доходы
			Вывести в консоль сумму чеков всех Покупателей из Списка покупателей
		Приватные методы:
		1) Выбрать категорию
			Выбрать категорию по ключу из Ассортимента
		2) Выбрать продукт
			Выбрать продукт из категории, выбранной с помощью Выбрать категорию
		Приватные свойства:
		1) Ассортимент
			Объект с массивами разных категорий продуктов
		2*) Список покупателей
			Массив экземпляров класса Покупатель, полученных методом Пришёл покупатель

	2*) Класс Покупатель, у которого будут:
		Публичные методы:
		1) Совершить покупку
			Получает параметром Ассортимент магазина, выбирает случайное количество товаров из случайного количества категорий, если его имеющегося количества хватает, добавляет их в Список покупок (количество продуктов в Ассортименте при этом должно уменьшаться)
		2) Показать список покупок
			Выводит названия продуктов из списка покупок
		3) Показать чек
			Выводит сумму цен купленных продуктов
		Приватные своства:
		1) Список покупок
			Массив с именами купленных продуктов
		2) Чек
			Массив с ценами купленных продуктов
