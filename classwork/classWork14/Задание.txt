1) В вашем IDE вам нужно установить какое-нибудь расширение "Live Server", потому что иначе из-за CORS вы не сможете запросить локальные файлы
    Далее ваш index.html откройте с помощью Live Server

2) Для кнопки "Загрузить текст" добавим функцию, что будет подгружать текст
    1) Создайте функцию loadText
    2) Создайте обработчик событий, который при клике по кнопке вызовет функцию loadText
    3) Внутри функции создайте XMLHttpRequest, который будет запрашивать файл "load-this.txt"
    4) После загрузки этого файла - используйте его содержимое как контент элемента "#text-wrapper" 

3) Для кнопки "Загрузить пользователей" добавим функцию, что будет подгружать текст
    1) Создайте функцию loadUsers
    2) Создайте обработчик событий, который при клике по кнопке вызовет функцию loadUsers
    3) Внутри функции создайте XMLHttpRequest, который будет запрашивать файл "my-users.json"
    5) Полученный JSON код нужно распарсить, используя методы стандартного JS объекта JSON.parse
    6) После загрузки этого файла - используйте его содержимое как контент элемента "#users-wrapper"
        Для каждого элемента в полученном объекте создайте дочерний элемент для <ul> по схеме
        <li id="user-{id}" class="{role}">{name}</li>

4) Перепешите прошлые два метода, используя fetch()

5) Перепешите прошлые два метода, используя jQuery.ajax()
