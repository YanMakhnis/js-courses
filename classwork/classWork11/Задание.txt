1) Установите (Если у вас не установлено) Node.js и NPM
    Ссылка на скачивание с официального сайта:
    https://nodejs.org/en/

    Для проверки успешной установки, запустите консоль и выполните две команды:
    node -v
    npm -v
    Если в ответ видите ошибку, что не известна команда node или npm - что-то пошло не так, либо нужно перезагрузить компьютер.

2) Создайте проект, используя Vite
    1) Запустите в консоли команду
    npm create vite@latest
    Следуйте этапам установочника. В проекте мы будем использовать React, VanilaJS
    2) Перейдите в консоли в папку созданного проекта и запустите его.
        Команда для перехода в папку 
        cd НАЗВАНИЕПАПКИ
        (cd -Change Directory. Название папки можно начать печатать и нажать Tab, тогда он сам допишет её название.)
        Команда для запуска
        npm run dev
    3) Откройте проект в браузере

3) Создаём проект:
	Выводим интерактивный список пациентов.

Файл patients.json содержит список пациентов.
В этом файле массив с 50 объектами.
Каждый объект - это информация о пациенте.
В объекте есть поля:
name - имя пациента. (Строка)
surname - фамилия пациента (Строка)
age -  возраст пациента (число)
gender - пол пациента (Строка 'M' или 'F')
kids - количество детей (число 0-4)
vaccinated - делал ли прививку (булевое значение)

	1) Сделайте вывод списка пациентов
	2) Сделайте вывод в виде html таблицы
		У таблицы будут столбцы:
		Имя, Фамилия, Возраст, Пол, Количество детей, Прививка от COVID-19
	3) Сортировка пациентов:
		При клике на заголовки таблицы пациенты будут сортироваться по выбранному полю.
		При клике по заголовку - сортировка по убыванию этого поля, повторный клик по этому же заголовку - сортировка по возрастанию этого поля и т.д.
	4) Над таблицей добавим форму фильтрации пациентов со следующими полями:
		Имя - текстовое (При вводе значений в этой строке, фильтруем пациентов, оставляя тех, у кого в Имени или Фамилии есть введёная подстрока)
		Минимальный возраст - числовое поле.(При вводе значений в этой строке, фильтруем пациентов, оставляя тех, у кого возраст больше или равен этому числу. Минимальное значение - 0, максимальное - вычисляется в начале, при рендере страницы, исходя из максимального имеющегося, либо значение, указанное в поле Максимальный возраст)
		Максимальный возраст - числовое поле.(При вводе значений в этой строке, фильтруем пациентов, оставляя тех, у кого возраст меньше или равен этому числу. Минимальное значение - 0, либо значение, указанное в поле Минимальный возраст, максимальное - вычисляется в начале, при рендере страницы, исходя из максимального имеющегося)
		Количество детей - числовое поле.(При вводе значений в этой строке, фильтруем пациентов, оставляя тех, у кого количество детей равно этому числу. Минимальное значение - 0, максимальное - вычисляется в начале, при рендере страницы, исходя из максимального имеющегося).
		Наличие прививки - чекбокс поле (При нажатом чекбоксе - фильтруем пациентов, оставляя тех, у кого это поле true. При отжатом значении - выводим всех.)
	5*) Используя SVG иконки, измените вариант вывода на страницу некоторых элементов, добавляя для них соответствующие иконки
