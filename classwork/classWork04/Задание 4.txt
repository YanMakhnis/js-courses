1) Напишите код, выполнив задание из каждого пункта отдельной строкой:
	1.1) Создайте пустой объект user.
	1.2) Добавьте свойство name со значением Виктория.
	1.3) Добавьте свойство surname со значением Толкачёва.
	1.4) Измените значение свойства name на Елена.
	1.5) Удалите свойство name из объекта.

2) Напишите функцию isEmpty(obj), которая принимает параметром какой-либо объект,
затем возвращает true, если у объекта нет свойств, иначе false.

3) Создайте объект salaries, который будет содержать некоторое количество пар
	ключ:значение. Ключами будет фамилия_имя человека, значением - его зарплата (число).
	3.1) Создайте функцию moneyAmount, которая будет возвращать сумму зарплат
	всех людей в данном объекте

4) В объекте есть свойство className, которое содержит список «классов» – слов, разделенных пробелом:
	4.1) Создайте функцию addClass(obj, cls), которая добавляет в список класс cls, но только если его там еще нет.
	4.2) Напишите функцию removeClass(obj, cls), которая удаляет класс cls, если он есть.
	4.3) Напишите функцию toggleClass(obj, cls), которая удаляет класс cls, если он есть, либо добавляет его, если такого класса не было.
	let someObj = { className: 'main desktop' }

5) Создайте объект car, объект представляет из себя экземпляр автомобиля. Изначально
	имеет свойства:
		brand (марка), значение будет строкой
		model (модель), значение будет строкой
		year (год выпуска), значение будет числом
		mileage (пробег, в километрах), значение будет числом
	Изначально все значения будут undefined
	Также, объект имеет методы:
		setBrand - принимает параметр. Если он является непустой строкой, то значение
				записывается в свойство brand
		setModel - принимает параметр. Если он является непустой строкой, то значение
				записывается в свойство model
		setYear - принимает параметр. Если он является числом, превышающим 1950,
				но меньше или равен текущему году, то оно записывается в свойство year
		increaseMileage - принимает параметр. Если он является положительным числом,
				то значение прибавляется в свойство mileage
				(если значение mileage отсутствовало, то принимает полученое значение)
		countAge - вызывается без параметров. Подсчитывает возраст автомобиля.
		showInfo - вызывается с необязательным параметром, строкой, которая указывает,
				какую информацию об автомобиле нужно вернуть.
				Если 'model', то модель автомобиля.
				Если 'brand', то марку автомобиля.
				Если 'year', то год выпуска автомобиля.
				Если 'age', то возраст автомобиля.
				Если 'mileage', то пробег автомобиля.
				Если значения у параметра нет, или любое другое, то всю информацию.

	5.1) Вызовите методы, чтобы заполнить все свойства объекта значениями, а после, проверьте работу метода showInfo, проверив все описанные кейсы его вызова.
