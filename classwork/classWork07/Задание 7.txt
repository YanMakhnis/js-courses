1) Создайте обработчик, следящий за кликами пользователя:
	1) Создайте функцию documentClickHandler
		1) Функция должна принимать аргументом объект события event
		2) Создайте переменные screenWidth и screenHeight, которые будут хранить ширину и высоту страницы в браузере
		3) Создайте переменные clickX и clickY. Используя свойства объекта event, определите координаты клика на странице, сохраните их в эти переменные.
			Важный момент, не путайте координаты клика в целом, относительно монитора, и относительно страницы сайта
		4) В зависимости от координат clickWidth и clickHeight, используя значение screenWidth и screenHeight, выведите в консоль, в какой части экрана был произведён клик, если условно его разделить на 4 зоны:
			"Лево-верх"
			"Право-верх"
			"Лево-низ"
			"Право-низ"
	2) Создайте переменную documentElement, которая выберет элемент <html>
	3) Добавьте обработчик событий для documentElement, который при клике будет вызывать documentClickHandler

2) Готовимся работать с картами, объект navigator
	1) Создайте переменную locationButton, найдите в DOM элемент, который при клике запустит функцию getUserData
	Создайте переменную locationValue, которая будет ссылкой на DOM элемент, в который запишите вычисленные координаты. 
	2) Создайте функцию getUserData
		1) Используя объект navigator, вызовите метод, позволяющий получить текущую геолокацию пользователя
		2) В качестве колбека, в случае успеха для этого метода передайте функцию userDataHandlerSuccess
		3) В качестве колбека, в случае провала для этого метода передайте функцию userDataHandlerFail
	3) Создайте функцию userDataHandlerSuccess
		1) Эта функция аргументом принимает данные, переданные ей
		2) Создайте переменную result, которая будет объектом
		3) Добавьте ей свойство language, которое с помощью объекта navigator будет отображать язык браузера пользователя
		4) Добавьте ей свойство latitude, которое из переданных функции данных определит долготу пользователя
		5) Добавьте ей свойство longitude, которое из переданных функции данных определит широту пользователя
		6) Выведите в консоль result
		7) В элемент locationValue, выведете значением `{latitude} {longitude}`
			После вызова функции скопируйте то, что вывел результат, откройте гугл или яндекс карты, введите скопированное значение, проверьте результат поиска
	4) Создайте функцию userDataHandlerFail
		1) Создайте переменную result, которая будет объектом
		2) Добавьте ей свойство language, которое с помощью объекта navigator будет отображать язык браузера пользователя
		3) Выведите в консоль result

3) Создайте функцию onPageLoad, которая будет запускаться при загрузки странице.
	1) Создайте переменную userStateWrapper, в неё поместите DOM элемент #user-state
	2) Добавить для document обработчик событий, который при срабатывании DOMContentLoaded запускает функцию onPageLoad
	3) При запуске функции onPageLoad проведите проверку, есть ли в Cookie сохранённая запись sessionId, со значением
		2.1) Если такое значение есть - в userStateWrapper замените текст на Авторизованный пользователь
		2.2) Если значения нет - вывести Confirm на экран, с вопросом "Хотите авторизоваться?".
		При положительном ответе - Нужно добавить запись sessionId в куки, со следующим значением и параметром, после чего перезагрузить страницу.
		Значение: одной строкой, без пробелов ГодМесяцЧислоЧасМинута
		Параметр: время жизни этой записи - 1 минута
