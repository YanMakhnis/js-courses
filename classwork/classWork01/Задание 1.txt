1) Подключение JS на странице
	1) Откройте файл index.html
	2) В теге body добавьте тег script, внутри которого добавьте команду "alert('Привет! Я работаю!')"
	3) Откройте index.html в браузере. Если вы увидели окошко с вашим сообщением - поздравляю!
	4) Теперь возвращаемся к редактированию файла, удалите или закомментируйте строку в файле.
		Не забывайте сохранять файл, после того, как произвели изменения.
	5) В теге head добавьте тег script и подключите файл script.js из папки scripts.
	6) Для проверки, удалось ли успешно подключить файл - откройте его и добавьте строку "alert('Давно не виделись! Здесь я тоже работаю!')"
	7) Проверьте в браузере, работает ли новый код. Если всё работает - вернитесь в файл и закомментируйте или удалите код.

После завершения каждого задания закомментируйте написанный код, чтобы он у вас остался, но при этом не запускался.

2) Создание переменных
	1) В файле script.js создайте несколько переменных, имена для переменных придумайте сами, в которые запишете следующую информацию:
		Имя, Фамилия, Пол
	2) Присвойте этим переменным значение
	3) Добавьте три комманды alert, которые выведут эти данные по очереди.

3) Взаимодействие с пользователем
	1) Создайте переменную
	2) Добавьте код, который будет спрашивать у пользователя его имя с помощью функции prompt.
	Это значение запишите в свою переменную.
	3) Выведите на экран сообщение с именем пользователя

4) Взаимодействие с пользователем 2.0
	1) Создайте переменную
	2) Добавьте код, который будет спрашивать у пользователя имя с помощью команды prompt,
	запишите это значение в вашу переменную
	3) Покажите сообщение с вопросом "Хотите проверить, верно ли мы запомнили ваше имя?"
	4) Если пользователь нажал Ok - покажите сообщение "{Имя}, верно?".
	Если нажал Отмена - вывести сообщение "Ладно, будем надеяться, что верно".
	5*) Если на 2 этапе пользователь не ввёл имя, либо оставил поле пустым, и нажал Ок, либо нажал Отмена - вывести сообщение "Ну зачем вы так?"
	Соответственно, в таком случае, пункты 3 и 4 не должны будут выполняться
