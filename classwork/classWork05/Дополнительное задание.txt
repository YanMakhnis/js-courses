1) Создайте несколько подклассов для Класса "Оружие":
	1) Пушка
	2) Автомат
	3) Лазер
2) Для каждого из этих классов добавьте приватное свойство "Вместительность обоймы".
	Свойство "Боеприпасы" по умолчанию имеет значением, как у свойства "Вместительность обоймы".
3) Добавьте приватный метод "Перезарядка"
	(Выводится в консоль сообщение "Я перезаряжаюсь". Значение "Боеприпасы" устанавливается равным "Вместительность обоймы")
4) Измените метод "Стрелять", если свойство "Боеприпасы" равно нулю - при вызове "Стрелять" вызываеся метод "Перезарядка.
5) Создайте несколько новых экземпляров трансформеров из разных фракций, вооружите их различными типами вооружения
6) Создайте подкласс для Класса "Оружие":
	Меч
