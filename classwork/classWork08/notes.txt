1. <meta name="robots" content="index,nofollow">
name="robots": Указывает поисковым системам, как индексировать страницу.
content="index,nofollow":
index: Позволяет поисковым системам индексировать страницу.
nofollow: Указывает, что ссылки на этой странице не должны передавать "вес" (или "ценность") другим страницам.

2. <meta name="description" content="Описание страницы">
name="description": Предоставляет краткое описание содержимого страницы.
content="Описание страницы": Это текст, который будет отображаться в результатах поиска под заголовком страницы. Он должен быть кратким и информативным.

информативным.
3. <meta name="keywords" content="ключевое, слово, другое, слово">
name="keywords": Указывает ключевые слова, связанные с содержимым страницы.
content="ключевое, слово, другое, слово": Список ключевых слов, разделенных запятыми. 

======BODY TAGS =====
Абстрактные:
<div> - Универсальный блочный элемент
<span> - Универсальный строчный элемент
Семантические (имеют смысловое значение и помогают браузерам, поисковым системам и разработчикам лучше понимать структуру контента):
<header> - Обычно содержит заголовки, логотипы, навигацию и другие вводные элементы.
<main>  - Основное содержимое страницы.
<footer> - Используется для создания подвала страницы или раздела.
<nav> - Используется для создания навигационных блоков (например, меню).

Элементы:
<h1>, <p>, <strong>,<i> - теги для текста
<a> ссылка
<ol>, <ul>, <dl> - списки
<table> - таблица
<form> - форма для отправки

Элементы формы:
<input> (аттрибут type, email, password(точками ввод), number, tel, url, search). Базовую валидацию можно аттрибутами добить.
