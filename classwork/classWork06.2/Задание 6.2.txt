1) Создать секундомер
	1) Создайте следующие переменные:
		1) timerWrapper - найдите в DOM элемент, который будет отображать значение таймера
		2) startButton - найдите в DOM элемент, который при клике запустит таймер
		3) stopButton - найдите в DOM элемент, который при клике остановит таймер
		4) resetButton - найдите в DOM элемент, который при клике обнулит значение таймера
		5) timerValue - будет хранить значение таймера
		6) timerId - индикатор таймера, для работы с ним
	2) Создайте следующие функции:
		1) startTimer - используя setInterval каждую секунду вызывает функцию increaseValue
			(добавьте проверку, чтобы не запустить таймер несколько раз)
		2) increaseValue - увеличьте значение timerValue, вызовите функцию showValue
		3) showValue - измените контент элемента timerWrapper, чтобы он показал текущее значение timerValue
		4) stopTimer - используя timerId остановит таймер запущенный startTimer
			Незабудьте сбросить значение timerId
			Добавьте проверку значения timerId, чтобы не пытаться остановить таймер, которого нет
		5) resetTimer - вызовите stopTimer, обнулите значение timerValue и вызовите showValue
	3) Добавьте обработчики событий для startButton, stopButton и resetButton, чтобы те вызывали соответствующую функцию
